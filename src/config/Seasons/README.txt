About these files/folders

SeasonsDataTemplate.dat contains setting that will be used in new worlds. Edit it with an NBT editor. The season and day will be randomized before it is used.
Settings.cfg contains basic settings for the mod. It can be edited with any text or word editor.
"Json Settings" (Settings.cfg's 'jsons' setting dependant) contains json-based settings for crops, biomes, etc.
